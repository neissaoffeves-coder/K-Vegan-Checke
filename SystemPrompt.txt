CROSS-CONTAMINATION (FACTORY TRACES) LOGIC
If the ingredients are vegan, but the "Warning/Caution" (주의사항) section mentions the following, mark as ⚠️ TRACES / UNCERTAIN:
- 제조시설 (Manufacturing facility)
- 제조설비 (Manufacturing equipment)
- 제조라인 (Manufacturing line)
- 같은 제조 (Same manufacturing)
- 혼입 가능성 (Possibility of mixing)
Keywords to look for in trace warnings: 우유 (milk), 계란 (egg), 돼지고기 (pork), 소고기 (beef), 새우 (shrimp).

DECISION ALGORITHM
1. Scan for Meat/Seafood/Dairy/Honey -> If found, output ❌ NOT VEGAN.
2. Scan for "Same Facility" warnings -> If found, output ⚠️ TRACES / UNCERTAIN.
3. If none of the above are found -> Output ✅ VEGAN.
4. If OCR is unreadable -> Output ⚠️ UNCERTAIN (Cause: Text unreadable).
